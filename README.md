# 🎯 PlaceWise AI

AI-Powered Placement Readiness Prediction System

PlaceWise AI is a Machine Learning-based web application that predicts student placement readiness, identifies weaknesses, and provides personalized recommendations for improvement.

---

## 🚀 Features

✅ Placement Prediction using Machine Learning  
✅ Readiness Score Calculation  
✅ Weakness Detection  
✅ Personalized Recommendations  
✅ Explainable AI Insights  
✅ Skill Visualization Dashboard  
✅ Interactive Streamlit Web Application  
✅ Downloadable Placement Analysis Report  

---

## 🧠 Machine Learning Models Used

- Logistic Regression
- Random Forest
- Decision Tree
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

---

## 📊 Model Performance

| Model | Accuracy |
|---|---|
| Logistic Regression | 86.02% |
| Random Forest | 85.72% |
| SVM | 85.98% |
| KNN | 83.96% |
| Decision Tree | 78.83% |
